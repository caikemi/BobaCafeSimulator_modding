Platform limitation: this CustomBGM example supports Windows only and is unavailable on Mac/macOS.

Put one or more .mp3 files directly in this directory.

After the game loads the Mod, it randomly plays songs from this directory. After one pass, it reshuffles the playlist and keeps looping.
If the directory contains no MP3 files, or none can be opened, the game falls back to the default background music implemented by the PlayerController Blueprint.

You may edit these values near the top of main.lua:
- priority: priority when several BGM Mods are enabled; larger numbers run first.
- SHUFFLE: true for random playback, false for filename order.
- VOLUME_MULTIPLIER: extra volume multiplier for this Mod; the game's music-volume setting still applies.
